# Explainable-AI-lab-3
# Лабораторная работа №3: Оценка неопределенности и визуализация в сверточных нейросетях

## Описание
Данная работа состоит из двух частей, демонстрирующих продвинутые методы анализа поведения нейронных сетей.

1.  **Оценка неопределенности (MC Dropout):** Исследование того, как метод Monte Carlo Dropout помогает выявить неуверенность модели при классификации зашумленных данных, что критически важно для систем, требующих высокой надежности (например, автопилот).
2.  **Визуализация принятия решений (Saliency vs. Grad-CAM):** Сравнение двух методов интерпретации работы CNN:
    *   **Vanilla Saliency:** Показывает, к каким пикселям (градиентам) модель наиболее чувствительна, выявляя текстуры и границы.
    *   **Grad-CAM:** Визуализирует, какие области изображения сыграли ключевую роль в классификации, подсвечивая целостные семантические области.
